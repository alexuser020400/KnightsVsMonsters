# Knights vs Monsters 🏰👹

A simple Java game featuring **Knights**, **Monsters**, and a **Player**, with both console and GUI modes.

## 🎮 Features
- Turn-based battles between Knights and Monsters
- Healing system
- Team statistics and victory conditions
- Color-coded map with Swing GUI
- Console mode for text-based gameplay

## 🚀 How to Run
Compile all Java files:
```bash
javac src/*.java
